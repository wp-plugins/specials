A handy & easy way to manage and display your company specials.

== Description ==

Your small business runs specials all the time and our plugin creates an easy way to showcase and manage those specials.

Simply add a title, short description (if needed) and an image. Then use our custom controls to position the text on your image.

**Some helpful plugins for more functionality**

Create New Images sizes for your Specials:<br />
http://wordpress.org/plugins/simple-image-sizes/

Reorder your specials faster:<br />
http://wordpress.org/plugins/cms-tree-page-view/



== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a specials page and include the [specials] shortcode on it.
1. Add your specials
